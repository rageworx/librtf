# librtf
A simple library for writing RTF

### Supported subsystems

* MinGW-W64

### Required
* MinGW-W64 G++

### How to build ?
* library:
    ```$ make``` at your shell.
* test:
    ```$ make``` in test directory, requires prebuilt librtf.a
